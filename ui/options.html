redirect to URL variable

[URL]

the current URL

[prevURL]

the user entered URL. In case of, user enter URL that contains redirect inside that page, the [prevURL] will be the URL before redirect.

For example, the page http://a.com has redirect script inside his own page to http://b.com.

When user enter http://a.com. The [URL] will be http://b.com and the [prevURL] will be http://a.com

[prevURLwoHttp]

prevURL but without http. i.e. if [prevURL] is http://google.com, [prevURLwoHttp] will be google.com
